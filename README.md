# KHU 다전공 시뮬레이터 (비공식)

경희대학교 다전공 학생을 위한 졸업요건 시뮬레이터입니다.  
인포21의 졸업사정 기능이 현재 이수 현황만 반영하는 점에서 착안해,  
**현재까지의 이수 상황 점검 + 미래 수강 계획 시뮬레이션**을 한 번에 할 수 있도록 개발하였습니다.

> 본 프로그램은 경희대학교 공식 서비스가 아닙니다.  
> 졸업 관련 최종 확인은 반드시 인포21 및 학과 사무실을 통해 하시기 바랍니다.

---

## 주요 기능

- 인포21 CSV 성적표 파싱 및 이수 현황 자동 분류
- 다전공 졸업요건 시뮬레이션 (전공기초 / 전공필수 / 전공선택)
- 미래 수강 과목 추가 및 학점 예측
- JSON 세이브 / 로드 기능
- 학기별 GPA 추이 시각화

## 실행 방법

### 요구 사항
- Python 3.8 이상 설치 필요 ([python.org](https://www.python.org/downloads/))

### Windows
1. ZIP 형태의 파일을 다운로드합니다 (Code → Download ZIP)
2. 압축 해제 후 폴더 안의 `run.bat` 실행
3. 자동으로 가상환경 생성 및 라이브러리 설치 후 브라우저가 열립니다

### 수동 실행
```bash
pip install -r requirements.txt
streamlit run app.py
```

## 파일 구조

```
khu-dual-major-simulator/
├── app.py              # 메인 애플리케이션
├── khu_planner.db      # 학과별 졸업요건 데이터베이스(다전공 구분)
├── sample_score.csv    # 샘플 성적표
├── requirements.txt    # 필요 라이브러리
└── run.bat             # Windows 실행 스크립트
```

## 사용 방법

1. 인포21 → 성적 → 전체성적조회 → 하단에 있는 전체성적보기(출력) 클릭 → CSV 형식으로 성적표 다운로드
2. 앱 실행 후 사이드바에서 `khu_planner.db` 업로드
3. 성적표 CSV 업로드
4. 교육과정 연도와 전공 선택 후 시뮬레이션 시작
